# gameDev

A Unity (C#) game development project.

## Requirements

- Unity 2022.3 LTS or newer
- Git

## Project Structure

```
Assets/
├── Scenes/      # Game scenes
├── Scripts/     # C# scripts
└── Prefabs/     # Reusable prefabs
ProjectSettings/ # Unity project settings
Packages/        # Unity package manifest
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/gameDev.git
   ```
2. Open the project in Unity Hub.
3. Select the Unity version and open the project.
