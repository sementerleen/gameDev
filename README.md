# gameDev

Unity tabanlı canavardan kaçış prototipi.

## Eklenenler
- Giriş ekranı akışı (`MainMenuController`)
- Yükleme ekranı akışı (`LoadingScreenController`)
- Sahne hedefini taşıyan yardımcı sınıf (`SceneLoadContext`)

## Hızlı kullanım
1. `UNITY_UI_SETUP.md` dosyasındaki adımlarla `Menu`, `Loading`, `Main` sahnelerini oluştur.
2. Build Settings'e sahneleri ekle.
3. Oyunu `Menu` sahnesinden çalıştır.
